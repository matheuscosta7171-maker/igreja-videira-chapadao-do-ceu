# Guia do Pastor

Após o acesso por link seguro, abra **Administração**.

- **Visão geral:** notificações e totais pendentes.
- **Cuidado pastoral:** pedidos de oração completos, solicitações de visita e seus estados.
- **Relatórios:** visão consolidada das células; permite revisar ou devolver relatórios.
- **Visitas:** confirme, solicite reagendamento, conclua ou recuse. O banco impede dois compromissos confirmados para o mesmo pastor, data e horário.

Pedidos, telefones, endereços e motivos de visita são privados. Não copie essas informações para canais públicos. As notificações internas funcionam sem WhatsApp; mensagens externas só funcionarão após configuração oficial e consentimento.
