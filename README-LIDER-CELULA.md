# Guia do Líder de Célula

1. Clique em **Área restrita**, informe seu e-mail cadastrado e abra o link recebido.
2. Entre em **Administração → Relatórios**. A célula é vinculada ao seu usuário pelo administrador.
3. Informe a data, membros, frequentadores, visitantes, ofertas, apelos e observações. Presença e oferta total são calculadas automaticamente também pelo banco.
4. Use **Salvar rascunho** para continuar depois ou **Enviar relatório** para concluir.
5. Relatórios enviados ficam bloqueados para alteração livre. Se forem devolvidos, corrija e envie novamente.
6. O gráfico mensal mostra somente os relatórios autorizados para seu perfil. Um líder não acessa dados de outra célula.

Valores de ofertas são privados e nunca aparecem no site público.
